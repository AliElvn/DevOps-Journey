# 📅 Day 05 - My First Linux Workspace

## 🎯 Goal

Set up my first Linux workspace and learn basic navigation commands.

---

## 📚 What I Learned

- WSL is only the platform; Ubuntu is a Linux distribution installed on top of it.
- Every Linux user has a Home Directory, represented by `~`.
- `pwd` shows my current working directory.
- `ls` lists the contents of the current directory.
- `cd` changes my current directory.
- `mkdir` creates a new directory.
- `touch` creates an empty file (or updates a file's timestamp if it already exists).
- Markdown (`.md`) is a lightweight markup language used for documentation.

---

## 💻 What I Did

- Installed Ubuntu on WSL.
- Explored my Home Directory.
- Created a `DevOps` workspace.
- Created `Projects`, `Notes`, and `Labs` directories.
- Created my first Linux file: `Day-01.md`.

---

## 💡 Biggest Lesson

Linux commands are tools that solve specific problems, not words to memorize.
